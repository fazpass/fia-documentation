# FIA Documentation (iOS)

Documentation on how to use FIA iOS SDK.

Choose the guide that fits your needs:

| Guide | Description |
|---|---|
| [Simple](README.IOS.SIMPLE.md) | Bare minimum setup to get started quickly. Uses a premade OTP view — no custom UI needed. |
| [Advanced](README.IOS.ADVANCED.md) | Full SDK configuration including Magic OTP and Magic Link setup. Uses a custom-made OTP view with manual auth type handling. |

# Check for User Verified Status

A successfully validated OTP does **not** mean that the user has also been successfully verified. To check the user's verified status, see the [Server Documentation](README.Server.md#check-for-user-verified-status).
